How QX AlphaRadar Works

Image of Data Flow Diagram

Traders
TG / Discord / Dashboard
  • EasyConnect listens to Qubic smart contract events (AddToBidOrder, IssueAsset, etc.).
  • All alerts are sent to a dedicated Make.com webhook.
  • The Make scenario routes events by type (buy, sell, liquidity, governance).
  • Each event branch formats a human-readable alert.
  • Alerts are broadcasted to Telegram/Discord and logged in Google Sheets.
  • This dashboard displays a simulated real-time view of these events.

About QX AlphaRadar

QX AlphaRadar is built by Naveen, an engineering student and trading enthusiast, for the Qubic | Hack the Future hackathon. It showcases how EasyConnect + Make + no-code automation can turn raw Qubic on-chain data into trader-friendly intelligence.
